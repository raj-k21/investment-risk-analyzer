# Investment Risk Analyzer

Built a financial analysis tool to evaluate stock performance using:

- Standard Deviation (Volatility)
- Beta (Market Sensitivity)
- Treynor Ratio (Risk-Adjusted Return)

## Features
- Calculates daily and monthly returns
- Computes volatility using standard deviation
- Estimates beta using regression
- Evaluates performance using Treynor Ratio

## Tools Used
- Excel
- Financial modeling concepts

## Concepts Applied
- Risk vs Return
- Systematic Risk (Beta)
- Risk-Free Rate
- Portfolio Analysis
